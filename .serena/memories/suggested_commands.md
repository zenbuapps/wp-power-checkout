# Suggested Commands

## Development
- `pnpm dev` - Start dev server
- `pnpm build` - Build for production
- `pnpm lint` - Lint all code
- `pnpm check-types` - TypeScript type checking

## Testing
- `cd tests/e2e && npx playwright test` - Run E2E tests

## Release
- `pnpm release` - Release patch version
- `pnpm release:minor` - Release minor version

## System Utils (Windows)
- `git`, `ls`, `cd`, `grep`, `find` via Git Bash
